# Scy

A Windows tool built with PowerShell and WPF. This project mainly exists as a way for me to learn more about Windows and PowerShell.

## What it does

A system utility thing for things like network config, managing SSH keys, notes, generating QR codes managing installed software and some tweaks.

## Requirements

- Windows 10/11
- PowerShell 5.1+
- Admin privileges (for some features)

## Running it

```
start_Scy.bat
```

or

```
Scy.vbs
```

or

```powershell
powershell -ExecutionPolicy Bypass -File Scy.ps1
```
